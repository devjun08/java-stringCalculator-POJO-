## :sparkles: 문제 설명 및 기능 요구사항 정리  
### 문제 설명 (핵심 기능 요구사항)
사용자가 입력한 문자열 값에 따라 사칙연산을 수행하는 계산기 구현

1. **입력형식**: 입력 문자열의 숫자와 사칙연산 기호 사이에는 공백이 포함되어야 합니다.   
2. **숫자처리**: 숫자는 쉼표(,) 또는 콜론(:)을 구분자로 가지는 문자열을 전달합니다.
3. **연산범위**: 연산자는 하나만 입력받으며 사칙 연산만 포함합니다.
4. **소숫점처리**: 소숫점의 경우 0.1, 0.2와 같이 소숫점 첫째자리까지만 표시합니다.
5. **예외 처리**: 사용자가 잘못된 값을 전달하면 'IllegalArgumentException'을 발생시킵니다.
___
## 입출력 예시
### 입력 예시
```bash
  
  input1: 1:2,3:4 +
  input2: 4:2,5 /
```

### 출력 예시
```bash
  
  output1: 1 + 2 + 3 + 4 = 10
  output2: 4 / 2 / 5 = 0.4
```
___
## Class 설계 및 구조

**stringCalculator**: 메인 진입점, 유효성 검사, 파싱 후 결과값을 반환합니다.  
**Validator**: 입력값의 유효성을 검사합니다. (null/빈 문자, 연산 기호 오 등)  
**Parser**: 입력 문자열을 분석하여 숫자 배열과 단일 연산로 분리합니다.  
**Operaotr**: 사칙연산(+, -, *, /) 로직을 담당합니다.
___
## 예외 처리 (IllegalArgumentException 발생 조건)
**입력 문자열 유효성**: 입력 문자열이 null 이거나 빈 문자열일 경우  
**숫자 유효성**: 숫자 문자열이 아닐 경우  
**연산자 유효성**: 입력된 연산 기호가 사칙연산이 아닌 경우
